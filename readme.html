  • Negra Export format: tiger_release_dec05.export (44 MB)

  • TIGER-XML format: tiger_release_dec05.xml (160 MB)

Documentation

  • A Brief Introduction to the TIGER Treebank: PS, PDF

  • TIGER Annotationsschema (German only): PS, PDF

  • Searching for Morphological Structure with Regular Expressions: PS, PDF

  • TIGER Morphologieannotationsschema (German only): PS, PDF

  • Indexing the TIGERCorpus (Working with the TIGERCorpus in the TIGERSearch software suite)

TIGERSearch

The TIGERSearch software suite is available for working with the TIGER Treebank. Users can query a corpus and browse through the graphical visualization of the corpus sentences. TIGERSearch is available at http://www.ims.uni-stuttgart.de/projekte/TIGER/TIGERSearch/.
